## Informations générales
Projet de gestion de listes de courses réalisé dans le cadre d'un TP pendant ma formation à l'[ENI ECOLE](https://www.eni-ecole.fr/).

## Technologies utilisées
* JavaSE-1.8
* Apache Tomcat v8.5
* Java Jersey (JAX-RS)
* Jackson
* Bootstrap 5.0.1
* jQuery 3.6.0
